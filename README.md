# libnidaqmx-mock

Mocking/Stubbing of libnidaqmx for unit testing of code using the library.

## Tools

- GCC
- GNU Make
- [Unity](https://github.com/ThrowTheSwitch/Unity)
- Docker (for cross-compilation to Windows target)

## Test

```
make test
```

## Build

```
make
```

## Installation

```
su
make install
```

## Build for Windows

```
./build.sh windows
```
